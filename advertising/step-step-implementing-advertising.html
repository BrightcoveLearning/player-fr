Vérifier la balise VAST

Pour vous assurer que le tag d'emplacement publicitaire VAST dont vous disposez est valide avec Google, procédez comme suit.

  1. Obtenez un tag d'emplacement publicitaire. (Si vous n'en avez pas, utilisez celui qui suit cette étape à des fins de démonstration.) Dans ce démarrage rapide, il est supposé que vous avez une annonce qui comprend une annonce pré-roll, mais toute annonce en ligne fonctionnera. 
    https://solutions.brightcove.com/bcls/brightcove-player/vmap/simple-vmap.xml
  2. Pour vérifier la validité de votre tag d'emplacement publicitaire, assurez-vous d'avoir copié son URL et accédez à la page suivante:

    Inspecteur de la suite vidéo (en cliquant sur ce lien, la page s'ouvrira dans une nouvelle fenêtre ou un nouvel onglet). Si vous utilisez l'URL fournie à l'étape 1, vous verrez que le tag d'emplacement publicitaire lit un pré-roll, un mid-roll à 5 secondes et un post-roll.

  3. Collez l'URL de votre tag d'emplacement publicitaire dans le Type d'entrée champ de saisie du formulaire.
  4. Cliquez sur TEST ANNONCE et vous verrez votre annonce jouer, suivie de la vidéo de course de moto fournie par Google.
  5. Assurez-vous que le test s'est terminé avec succès, puis passez à la section suivante. Si le test échoue, vous devez vérifier votre tag d'emplacement publicitaire.
Remarque : Le tag d'emplacement publicitaire VAST peut être hébergé par votre fournisseur d'annonces, ou vous pouvez copier le XML et l'héberger sur votre propre URL.

Activation des spots publicitaires

Brightcove Player s'appuie sur le plug-in IMA3 pour lire les publicités VAST. Pour ajouter ledit plugin à un lecteur, suivez ces étapes.

  1. Connectez-vous à Studio.
  2. Ouvrez le JOUEURS module et soit créer un nouveau lecteur, soit localiser le lecteur auquel vous souhaitez ajouter des fonctionnalités publicitaires.
  3. Cliquez sur le lien du lecteur pour ouvrir les propriétés du lecteur.
  4. Cliquez sur Publicité dans le menu de navigation de gauche.
  5. Vérifier la Activer côté client (IMA).
  6. Pour le Tag publicitaire , saisissez le tag d'emplacement publicitaire ci-dessus. Pour les tests, vous pouvez utiliser les éléments suivants: 
    https://solutions.brightcove.com/bcls/brightcove-player/vmap/simple-vmap.xml
  7. Laissez les autres valeurs par défaut. Votre La publicité La section doit apparaître comme suit:
  8. Cliquez sur Save.
  9. Pour publier le lecteur, cliquez sur Publier et intégrer > Publier les modifications.
  10. Pour fermer la boîte de dialogue ouverte, cliquez sur Fermer.

Publier une vidéo dans un lecteur

Vous publiez une vidéo dans le lecteur nouvellement créé dans le module Media. Une fois que vous avez associé la vidéo à votre lecteur, vous publiez la vidéo et aurez ensuite accès à trois implémentations pour visionner la vidéo dans votre lecteur.

Pour publier une vidéo sur votre lecteur, procédez comme suit:

  1. Dans Studio, accédez au module Médias en sélectionnant MÉDIAS dans le menu principal déroulant.
  2. Sélectionnez la vidéo que vous souhaitez publier sur votre lecteur. Dans la rangée supérieure de boutons, sélectionnez Publier et intégrer et alors Lecteur Web.
  3. À partir du contrôle de sélection, dans le Sélectionner un joueur section de la page, choisissez votre lecteur nouvellement créé.
  4. Vous avez désormais accès aux trois implémentations du player, Aperçu dans le navigateur , la norme et Avancée. Ne fermez pas cette fenêtre car vous cliquerez sur un lien de celle-ci.
Remarque : Si vous êtes curieux, vous pouvez voir « sous le capot » de la configuration du lecteur à l'aide du code URL. Clique sur le Aperçu dans le navigateur code lien, puis copiez et parcourez l'URL générée. Ensuite, changez le index.html à la fin de l'URL vers config.json. Vous verrez un objet JSON détaillant la configuration actuelle du lecteur que vous venez de créer.

Attribuer une vidéo au lecteur

Les propriétés du contenu vidéo sont utilisées pour attribuer du contenu vidéo au lecteur.

Pour configurer les propriétés du contenu vidéo, procédez comme suit.

  1. Allez dans la section Video Content et cliquez sur Edit.
  2. Ajoutez le ou les rendus de votre choix.
  3. Ajoutez une image d'affiche. Si vous n'ajoutez pas d'image d'affiche, la première image de la vidéo sera utilisée.
  4. Ne définissez aucune dimension pour la vidéo car cela se fera via CSS dans la page.
  5. Cliquez sur Save.
  6. Suivez le processus pour publier votre lecteur.

Test

Bien sûr, vous voudrez vous assurer que l'annonce fonctionne dans votre lecteur.

Pour voir que votre ajout joue dans votre lecteur, procédez comme suit.

  1. Parmi les trois implémentations, cliquez sur le Aperçu dans le navigateur lien.
  2. Lisez la vidéo et vous verrez vos annonces lues, suivie de la vidéo que vous avez affectée au lecteur.

Bien sûr, vous voudrez être sûr que l'annonce fonctionne dans votre joueur. Dans ce cas, vous testerez en utilisant la version d'aperçu du lecteur, il n'est donc pas nécessaire d'attendre que le lecteur soit réellement publié. Bien que le temps de publication ne soit pas long, l'utilisation de la version préliminaire du lecteur élimine toute confusion de quelque chose qui ne fonctionne pas parce que le lecteur n'a pas terminé la publication.

Pour voir votre ajout jouer dans votre lecteur, procédez comme suit.

  1. En haut de la page, sélectionnez Code intégré et URL puis Aperçu du lecteur.
  2. Clique le URL du joueur.
  3. Lisez la vidéo et vous verrez vos annonces lues, suivies de la vidéo que vous avez attribuée au lecteur.
