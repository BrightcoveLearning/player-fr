Test des plugins publicitaires pré-version

{{ page.description }}

Présentation

La diffusion d'annonces vidéo dans Brightcove Player peut être une tâche complexe. Plusieurs entités (Brightcove Player lui-même, un serveur publicitaire et le plugin publicitaire spécifique) sont toutes impliquées dans la diffusion réussie de la publicité. Pour fournir un environnement de test pour les mises à jour des plugins IMA3 et FreeWheel, Brightcove mettra à disposition des versions préliminaires des plugins. Ce document fournit des instructions sur la façon de tester avec ces versions préliminaires.

Les pré-versions seront annoncées dans le Forum des développeurs Brightcove Web Player , ainsi que dans le changelog sections du Publicité avec le plugin IMA3 et Publicité avec le plugin FreeWheel documents.

Tester IMA3 dans Studio

Les étapes suivantes vous guident sur la façon de tester un plug-in IMA3 pré-version dans Studio:

  1. Pour des raisons de sécurité, vous devez enregistrer la version actuelle / plugins / plugin-version-reference.htmls du plugin que vous utilisez actuellement. Cela vous permettra de restaurer la version du plugin que vous utilisiez avant de tester. Vous pouvez toujours trouver la dernière version du plugin dans le Référence de version du plug-in document.
  2. Ouvrez le JOUEURS module et soit créer un nouveau lecteur, soit localiser le lecteur auquel vous souhaitez ajouter le plugin.
  3. Cliquez sur le lien du lecteur pour ouvrir les propriétés du lecteur.
  4. Cliquez sur Publicité dans le menu de navigation de gauche.
  5. Décochez la case Activer côté client (IMA). Vous ne voulez pas que cette option soit activée car elle pourrait interférer avec la version que vous allez bientôt charger manuellement.
  6. Cliquez sur Save.
  7. Cliquez sur Plugins dans le menu de navigation de gauche.
  8. Ensuite, cliquez sur Ajouter un plugin.
  9. Pour le nom du plugin, entrez ima3.
  10. Pour le JavaScript URL, saisissez l'URL du fichier JavaScript de pré-version.
  11. Pour le CSS URL, saisissez l'URL du fichier CSS de pré-version.
  12. Entrez les options de configuration dans le Options (JSON) zone de texte. Si vous avez précédemment configuré le plugin IMA3 à l'aide du Plugins section, vous n'aurez probablement pas besoin de modifier les options.
  13. Cliquez sur Save.
  14. Pour utiliser le lecteur, cliquez sur Publier et intégrer. Vous pourrez tester à l'aide du lecteur Aperçu, ou vous pouvez publier et utiliser le lecteur publié.
  15. Pour fermer la boîte de dialogue ouverte, cliquez sur Fermer.

Tester FreeWheel dans Studio

Les étapes suivantes vous guident sur la façon de tester un plugin FreeWheel pré-version dans Studio:

  1. Pour des raisons de sécurité, vous devez enregistrer les versions actuelles du plugin que vous utilisez actuellement. Cela vous permettra de restaurer la version du plugin que vous utilisiez avant de tester. Vous pouvez toujours trouver la dernière version du plugin dans le Référence de version du plug-in document.
  2. Ouvrez le JOUEURS module et soit créer un nouveau lecteur, soit localiser le lecteur auquel vous souhaitez ajouter le plugin.
  3. Cliquez sur le lien du lecteur pour ouvrir les propriétés du lecteur.
  4. Cliquez sur Plugins dans le menu de navigation de gauche.
  5. Ensuite, cliquez sur Ajouter un plugin OU si vous voyez FreeWheelPlugin dans la liste des plugins, cliquez dessus.
  6. Pour le nom du plugin, entrez FreeWheelPlugin.
  7. Pour l'URL JavaScript , entrez l'URL du fichier JavaScript pré-version.
  8. Pour le CSS URL, saisissez l'URL du fichier CSS de pré-version.
  9. Entrez les options de configuration dans la zone de texte Options (JSON) . Si vous avez précédemment configuré le plugin FreeWheel à l'aide du Plugins section, vous n'aurez probablement pas besoin de modifier les options.
  10. Cliquez sur Save.
  11. Pour utiliser le lecteur, cliquez sur Publier et intégrer. Vous pourrez tester à l'aide du lecteur Aperçu, ou vous pouvez publier et utiliser le lecteur publié.
  12. Pour fermer la boîte de dialogue ouverte, cliquez sur Fermer.

Tester la mise en œuvre programmatique

Le test où vous avez implémenté le plugin par programme est le même que vous utilisiez le plugin IMA3 ou FreeWheel. En gros, vous remplacez simplement les chemins d'accès existants aux fichiers JavaScript et CSS par ceux de la version préliminaire. Les étapes pour effectuer cette tâche sont les suivantes.

  1. Pour des raisons de sécurité, vous devez enregistrer les versions actuelles du plugin que vous utilisez actuellement. Cela vous permettra de restaurer la version du plugin que vous utilisiez avant de tester. Vous pouvez toujours trouver la dernière version du plugin dans le document de référence de version du plugin .
  2. Dans votre code, recherchez le <link> et <script> balises qui intègrent le CSS et JavaScript pour le plugin approprié. Dans le code ci-dessous, des exemples pour le plugin IMA3 sont mis en évidence.
  3. Remplacez ces chemins par les versions préliminaires.
  4. Actualisez votre page et vous utiliserez la version préliminaire.
  5. Une fois le test terminé, modifiez à nouveau les chemins d'accès aux fichiers JavaScript et CSS en utilisant les chemins enregistrés pour restaurer votre configuration d'origine.
