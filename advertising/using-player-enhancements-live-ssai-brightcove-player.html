Utilisation des améliorations de lecteur pour Live SSAI avec Brightcove Player

{{ page.description }}

introduction

Les améliorations du lecteur pour le SSAI en direct de Brightcove vous permettent d'améliorer les pauses publicitaires dans les flux en direct avec le nombre d'annonces, les minuteries de compte à rebours, les annonces de clics et les annonces complémentaires. Ces flux en direct peuvent être lus à l'aide de Brightcove Player.

Si vous êtes nouveau dans cette fonctionnalité, consultez les points suivants :

  • Aperçu: Améliorations du lecteur pour le SSAI en direct
  • Implémentation des améliorations de lecteur pour Live SSAI

Étapes

Pour lire des flux en direct avec des améliorations de lecteur à l'aide du lecteur Brightcove, procédez comme suit:

  1. Créez ou choisissez un lecteur existant dans Video Cloud Studio Joueurs module.

  2. Brightcove Player doit être configuré avec le plugin SSAI. Suivre la Ajouter le plugin SSAI étapes décrites dans le Implémentation de SSAI par programmation section du document du plugin SSAI.

    Pour les améliorations du lecteur avec Live SSAI, Brightcove Player nécessite le plug-in SSAI (videojs-ssai). Actuellement, vous ne pouvez pas utiliser Video Cloud Studio pour ajouter le plug-in SSAI à votre lecteur, car il nécessite l'ID de configuration de l'annonce.
  3. Suivez le flux de travail dans le Implémentation des améliorations du lecteur pour Live SSAI document, jusqu'à la création de votre jeton de lecture.
  4. Choisissez une implémentation:
    • Code intégré standard
    • Code d'intégration avancé
    • Source SSAI en direct
  5. Continuez avec le flux de travail dans le document Implémentation Player Améliorations for Live SSAI .

Code intégré standard

Lors de la publication d'une vidéo à partir de Video Cloud Studio Médias module, vous pouvez utiliser le code d'intégration standard pour lire votre flux en direct. Suivez ces étapes :

  1. Du Médias module, sélectionnez votre vidéo à distance pour la diffusion en direct et publiez-la avec un lecteur compatible SSAI.
  2. Copiez le code d'intégration standard et collez-le dans votre page Web.

  3. dans le <iframe> tag, vous devriez voir un src attribut. À la fin de cela, ajoutez le adConfigId Paramètre d'URL avec la valeur de votre jeton de lecture.

    <iframe src="https://players.brightcove.net/6027103979001/IqljF4Olr_default/
index.html?videoId=6089818309001&adConfigId=live.aaB61234aabbccdd567890"
  allowfullscreen=""
  allow="encrypted-media"
  width="960" height="540">
</iframe>
  4. Votre joueur est prêt à lire votre diffusion en direct.

Code d'intégration avancé

Lors de la publication d'une vidéo à partir de Video Cloud Studio Médias module, vous pouvez utiliser le code d'intégration avancé pour lire votre flux en direct. Suivez ces étapes:

  1. Dans le module Media , sélectionnez votre vidéo à distance pour la diffusion en direct et publiez-la avec un lecteur compatible SSAI.
  2. Copiez le code intégré avancé et collez-le dans votre page Web.

  3. dans le <video-js> tag, ajoutez un data-ad-config-id attribut avec la valeur de votre jeton de lecture.

    <video-js
  data-account="6027103979001"
  data-player="IqljF4Olr"
  data-embed="default"
  controls=""
  data-video-id="6089818309001"
  data-ad-config-id="live.aaB61234aabbccdd567890"
  data-playlist-id=""
  data-application-id=""
  width="960" height="540">
</video-js>
<script src="https://players.brightcove.net/6027103979001/IqljF4Olr_default/index.min.js"></script>
  4. Votre joueur est prêt à jouer votre flux en direct.

Source SSAI en direct

Contrairement aux sources VOD SSAI, les sources Live SSAI n'incluent pas de VMAP initial. En effet, les flux SSAI en direct ne savent pas à l'avance quand les coupures publicitaires seront incluses. Au lieu de cela, les coupures publicitaires sont déclenchées à partir de points de repère que vous ajoutez à la tâche en direct.

Les flux Brightcove Live SSAI commencent par une URL de manifeste HLS standard avec un schéma de dénomination spécifique:

https://.../playlist_ssaiM.m3u8

Le plugin SSAI recherche le _ssaiM.m3u8 suffixe et le type MIME HLS normal

Lors de la publication d'une vidéo à partir de Video Cloud Studio Médias module, vous pouvez lire un flux SSAI en direct comme suit:

  1. Dans le module Media , sélectionnez votre vidéo à distance pour la diffusion en direct et publiez-la avec un lecteur compatible SSAI.
  2. Copiez le code intégré avancé et collez-le dans votre page Web.

  3. À votre code de page, ajoutez le code suivant:

    <video-js id="myPlayerID"
  data-account="6027103979001"
  data-player="IqljF4Olr"
  data-embed="default"
  controls=""
  data-video-id=""
  data-playlist-id=""
  data-application-id=""
  width="960" height="540">
</video-js>
<script src="https://players.brightcove.net/6027103979001/IqljF4Olr_default/index.min.js"></script>
<script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
    myPlayer.src({
      type: 'application/x-mpegURL',
      src: 'https://path/to/playlist_ssaiM.m3u8'
    })
  });
</script>
  4. Votre joueur est prêt à jouer votre flux en direct.
