Notions de base des développeurs de joueurs : Plugin personnalisé - Utilisation de Studio

Dans cette rubrique, vous apprendrez comment attribuer un plugin personnalisé à un lecteur Brightcove à l'aide de Studio.
 

Étapes

  1. Dans Studio, sélectionnez le JOUEURS module.
  2. Cliquez sur le lecteur auquel vous souhaitez ajouter le plugin personnalisé.
  3. Dans les propriétés du lecteur, sur le côté gauche, sélectionnez le Plugins option.
  4. Du Ajouter un plugin liste déroulante, sélectionnez Plugin personnalisé.
  5. Ajouter ce qui suit :
    • Nom du plugin
    • URL JavaScript
    • URL CSS - si nécessaire
    • Options (JSON) - si nécessaire
  6. Sauvegardez le joueur.
  7. Publier le joueur.
