Notions de base des développeurs de joueurs : Génération du code de joueur

Dans cette rubrique, vous apprendrez comment générer du code de joueur que vous pourrez utiliser ultérieurement dans le développement du lecteur.
  1. Dans Studio, ouvrez le JOUEURS module.
  2. Cliquez sur le bouton New Player.
  3. Donnez le joueur et son nom puis cliquez sur sauver.
  4. Ouvrez le MÉDIAS module.
  5. Cliquez sur une vidéo pour publier.
  6. Dans la page des propriétés du lecteur, cliquez sur le bouton Publier et intégrer bouton, dans le menu déroulant, sélectionnez Lecteur Web .
  7. Pour le joueur réel, sélectionnez le joueur que vous venez de créer.
  8. dans le Copier le code section sélectionnez le Avancée la mise en oeuvre.
  9. Définissez ce qui suit :
    • Taille: Fixé
    • Ratio d'aspect: Faites correspondre votre vidéo
    • Largeur du joueur: Largeur souhaitée
  10. Enfin, copiez le code.
