Notions de base des développeurs de joueurs : Préparation à l'écriture du code

Dans cette rubrique, vous apprendrez à créer un modèle de bonnes pratiques standard qui vous prépare à écrire du code pour contrôler le lecteur.

Étapes

  1. Collez une instance du code du lecteur d'implémentation avancée dans une page HTML. 
    <video data-video-id="5831704295001"
  data-account="921483702001"
  data-player="Uj7Yz80yM"
  data-embed="default"
  data-application-id
  class="video-js"
  controls
  width="640"
  height="360"></video>
<script src="//players.brightcove.net/921483702001/Uj7Yz80yM_default/index.min.js"></script>
  2. Ajouter un id attribut au <video> tag et attribuez-lui la valeur myPlayerID. 
    <video id="myPlayerID"
  3. Ajouter un <script> bloc juste au-dessus du </body> marque.
  4. dans le script block, utilisez le videojs.getPlayer() méthode pour obtenir une référence au lecteur sur la page. Utilisez le id ajouté plus tôt comme argument de la méthode. 
    <script>
  videojs.getPlayer('myPlayerID')
</script>
  5. Utilisez le ready() et codez une fonction de rappel anonyme comme argument. 
    <script>
  videojs.getPlayer('myPlayerID').ready(function() {

  )};
</script>
  6. Déclarez une variable, nommée myPlayer dans la fonction anonyme et attribuez-lui le mot-clé this , qui représente le joueur dans le contexte de la fonction anonyme. 
    var myPlayer = this;
