AddChild (enfant, [options])

Ajoute un composant enfant à l'intérieur de ce composant

MyComponent.el () ;//-> <div class='my-component'></div > MyComonent.Children () ;//[tableau vide] var MyButton = MyComponent.AddChild ('MyButton') ;//-> <div class='my-component'><div class="mon-bouton">MyButton<div></div >//-> MyButton === MyComonent. enfants () [0] ;

Transmettez les options pour les constructeurs enfants et les options pour les enfants de l'enfant

var MyButton = MyComponent.AddChild ('MyButton', { text: 'Press Me', children: { buttonChildExample: { buttonChildOption: true }}}) ;
PARAMÈTRES :
  • child string|VJS.Component Le nom de classe ou l'instance d'un enfant à ajouter
  • objets options (FACULTATIF) Options, y compris les options à transmettre aux enfants de l'enfant.
RETOURS :
  • VJS.Component Le composant enfant (créé par ce processus si une chaîne a été utilisée)

hauteur ([num], [SkiPlisteners])

Obtenir ou définir la hauteur du composant (valeurs CSS)

La définition des valeurs de dimension de balise vidéo ne fonctionne qu'avec des valeurs exprimées en pixels. Les valeurs en pourcentage ne fonctionneront pas. Certains pourcentages peuvent être utilisés, mais width () /height () renvoie le nombre +%, et non la largeur/hauteur calculée réelle.

PARAMÈTRES :
  • num Number|Chaîne (FACULTATIF) Nouvelle hauteur de composant
  • SkiListeners Boolean (FACULTATIF) Ignorer le déclencheur d'événement de redimensionnement
RETOURS :
  • VJS.Component Ce composant, lors de la définition de la hauteur
  • Number|String La hauteur, lors de l'obtention

options (obj)

Fusion profonde des objets d'options

Chaque fois qu'une propriété est un objet sur les deux objets options, les deux propriétés sont fusionnées à l'aide de VJS.obj.DeepMerge.

Il est utilisé pour fusionner les options des composants enfants. Nous voulons qu'il soit facile de remplacer les options individuelles d'un composant enfant sans avoir à réécrire toutes les autres options par défaut.

Parent.Prototype.Options_ = { children: { 'childOne': { 'foo': 'bar', 'asdf': 'fdsa' }, 'ChildTwo' : {}, 'ChildThree' : {}}} NewOptions = { children: { 'childOne': { 'foo': 'baz', 'abc': '123' } 'ChildTwo' : null, 'ChildFour' : {} }} this.options (nouvelles options) ;

RÉSULTAT

{ children: { 'childOne': { 'foo': 'baz', 'asdf': 'fdsa', 'abc': '123' }, 'ChildTwo' : null,//Désactivé. Ne sera pas initialisé. 'ChildThree' : {}, 'ChildFour' : {}}}
PARAMÈTRES :
  • Obj Object de nouvelles valeurs d'options
RETOURS :
  • Objet Un nouvel objet de ce .options_ et obj fusionné
