Accessibilité du lecteur Brightcove

{{ page.description }}

Commandes du clavier

Le lecteur Brightcove peut être contrôlé avec le clavier. En tabulant sur les commandes du joueur, les touches suivantes peuvent être utilisées pour contrôler et manipuler le joueur :

  • Gros bouton de lecture : appuyez sur Espace ou sur Entrée pour lancer la lecture.
  • Bouton de lecture dans la barre de commandes : appuyez sur Espace ou sur Entrée pour alterner entre lecture et pause.
  • Barre de progression : utilisez les touches haut ou droite pour avancer de cinq secondes, utilisez les touches bas ou gauche pour revenir cinq secondes en arrière.
  • Plein écran : appuyez sur Espace ou sur Entrée pour alterner entre plein écran et fenêtré.
  • Curseur de volume : utilisez les touches haut ou droite pour augmenter le volume, utilisez les touches bas ou gauche pour baisser le volume.

Lecteurs d'écran

Le Brightcove Player est compatible avec les lecteurs d'écran des navigateurs de bureau ainsi que des appareils iOS et Android.

Les tests les plus récents sont réalisés selon les normes WCAG 2.0 AA avec les configurations suivantes :

  • TalkBack sur Android
  • VoiceOver sur iOS 9+
  • Narrateur Windows avec IE10+
  • La dernière version de JAWS sous Windows avec Firefox et Chrome sous Windows
  • Mac OS VoiceOver avec Safari, Chrome et Firefox
Lorsque le lecteur d'écran JAWS est utilisé avec le navigateur Google Chrome, les commandes au clavier peuvent ne pas fonctionner comme prévu. Maintenir la touche Maj tout en utilisant les commandes du clavier peut soulager le problème. Cependant, il est recommandé d'utiliser Firefox avec JAWS à la place.
