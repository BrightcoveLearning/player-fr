    Identifier et supprimer des vidéos peu performantes

    Dans cette rubrique, vous apprendrez à utiliser Video Cloud Studio et une feuille de calcul pour identifier et supprimer les vidéos peu performantes.

    Introduction

     

    Pour un certain nombre de raisons, il est probablement préférable de ne pas avoir de vidéos dans votre bibliothèque qui ne sont pas vues, et vous pensez qu'elles ne seront pas non plus vues à l'avenir. Dans ce document, vous verrez comment identifier ces vidéos et les supprimer de votre bibliothèque. Les puces suivantes fournissent un aperçu de haut niveau du processus. Les étapes détaillées suivent ensuite.

    • Utilisez le module Média pour exporter un fichier contenant une ligne d'informations pour chaque vidéo de votre bibliothèque.
    • Utilisez le module Analytics pour créer un rapport personnalisé qui inclut l'ID vidéo, le titre, les impressions vidéo et les vues vidéo. Le rapport sera exporté au format CSV (valeurs séparées par des virgules) et répertoriera uniquement les vidéos avec des vues et / ou impressions vidéo différentes de zéro.
    • Importez les deux fichiers CSV dans une feuille de calcul Google Docs.
    • Ajoutez une colonne à la feuille de calcul de toutes les vidéos qui détermine si la vidéo détaillée dans une ligne particulière a des vues vidéo ou non. Cela se fait en vérifiant si un identifiant dans la feuille de calcul Toutes les vidéos a une entrée correspondante dans la feuille de calcul du rapport Analytics.
    • Copiez les identifiants des vidéos sans vues dans un document Google et manipulez le format à utiliser dans un processus de suppression groupée.
    La suppression de vidéos est une opération drastique et il n'est pas possible de les restaurer. Pour des raisons de sécurité, assurez-vous d'avoir une copie locale d'une vidéo avant de la supprimer.

    Exporter toutes les données des vidéos

    Du Médias module, vous pouvez obtenir un rapport sur tous les médias de votre bibliothèque. Cela sera utilisé plus tard dans une feuille de calcul.

    1. Entrez dans le studio Médias module.
    2. Clique sur le Exportation bouton près du haut de la page.
    3. Dans la boîte de dialogue qui s'ouvre, assurez-vous Toutes les vidéos de la liste est sélectionné.
    4. Clique le Exporter des données bouton. Notez que ce rapport exportera les informations pour un maximum de 15 000 vidéos.
    5. Une fois le rapport terminé, cliquez sur le bouton Télécharger un fichier bouton.
    6. Notez que le nom de fichier correspond à votre ID de compte et que le fichier est au format CSV (valeurs séparées par des virgules).
    7. Cliquez sur Annuler pour fermer la boîte de dialogue.

    Créer un rapport Analytics personnalisé

    Pour créer le rapport Analytics qui inclut des vidéos avec vues, procédez comme suit:

    1. Dans Studio, accédez au ANALYTIQUE module.
    2. Dans le menu de gauche, cliquez sur Douane.
    3. Utilisez le Inclure les comptes menu déroulant pour choisir le ou les comptes souhaités pour le rapport.
    4. Choisir la Type de rapport être One-time.
    5. Sélectionner un Plage de dates pour le rapport à l'aide des calendriers.
    6. Pour le Groupes de données choisir Video.
    7. Pour le Champs de données choisissez les trois champs Video Name , Video Views et Video Impressions:
    8. Sous les champs, vous voyez un aperçu de votre rapport. Notez que l'ID vidéo est automatiquement inclus dans le rapport:
    9. Fournir un Nom du rapport ou acceptez la valeur par défaut.
    10. Choisir la Format être CSV.
    11. Choisissez de recevoir un e-mail ou non lorsque le rapport est prêt. Il est fort probable que vous attendiez simplement le rapport, sauf si votre compte contient un très grand nombre de vidéos.
    12. Clique le Demander un rapport bouton.
    13. Après un certain délai, vous verrez que le rapport est téléchargé.

    Importez le fichier de toutes les vidéos

    Dans ce document, Google Docs sera utilisé car la fonctionnalité est accessible à tous gratuitement. Si vous êtes un utilisateur d'Excel, vous pouvez l'utiliser pour effectuer toutes les tâches de feuille de calcul présentées dans ce document.

    Dans cette section du document, vous ouvrirez le rapport de toutes les vidéos exportées créé à partir du module Médias.

    1. Ouvrez une feuille de calcul vierge Google Docs.
    2. Sélectionner Fichier> Importer> Télécharger , puis faites glisser le fichier CSV de toutes les vidéos (nommé d'après votre ID de compte) vers l'emplacement approprié ou parcourez-le pour le sélectionner.
    3. Sur le Importer le fichier boîte de dialogue, acceptez les valeurs par défaut et cliquez sur Importer des données:
    4. Vos données importées ressembleront à ce qui suit:

    Importer le rapport Analytics

    Vous allez maintenant importer dans une autre feuille (dans le même groupe de feuilles de calcul) le rapport Analytics que vous avez créé.

    1. Sélectionner Fichier> Importer> Télécharger , puis faites glisser le fichier CSV du rapport Analytics vers l'emplacement approprié ou recherchez-le pour le sélectionner.
    2. Sur le Importer le fichier , ASSUREZ-VOUS de changer Importer l'emplacement à Insérer de nouvelles feuilles. Acceptez les autres valeurs par défaut et cliquez sur Importer des données:
    3. Vos données importées ressembleront à ce qui suit:
    4. Notez qu'au bas de la feuille de calcul, vous voyez les deux feuilles répertoriées.

    Créer une plage nommée

    Vous allez maintenant ajouter une colonne qui déterminera si une vidéo a des vues. Pour ce faire, vous devrez référencer les ID dans le rapport Analytics (les vidéos avec vues). Pour faciliter le processus, vous allez maintenant créer et nommer une plage.

    1. Tout en restant dans la feuille de rapport Analytics, cliquez sur la majuscule UNE en haut de la première colonne.
    2. Faites un clic droit quelque part dans la colonne et sélectionnez Définir la plage nommée ...
    3. Un panneau s'ouvrira sur le côté droit de la feuille de calcul. Changer le nom par défaut de NamedRange1 à IDsWithViews.
    4. Après avoir appuyé sur Entrée, vous verrez la plage définie.

    Ajouter une colonne pour la détermination des non vues

    Vous allez maintenant ajouter une colonne et ajouter un VLOOKUP() fonction qui signalera si une vidéo spécifique a des vues.

    1. Revenez à la feuille de toutes les vidéos, nommée avec votre identifiant de compte.
    2. Cliquez sur la colonne B.
    3. Dans le menu de la feuille de calcul, sélectionnez Insérer> Colonne à gauche. Cela crée une nouvelle colonne B vide.
    4. Cliquez dans la cellule B2.
    5. Entrer =VLOOKUP(. Cela démarre l'utilisation de la fonction.
    6. Juste après les parenthèses d'ouverture, cliquez sur le A2 cellule. Cet emplacement de cellule sera inséré. Cela définit ce que la fonction recherchera.
    7. Entrez une virgule après la cellule, puis tapez id. Vous verrez que vous pouvez sélectionner la plage nommée et la sélectionner. Cela définit où la fonction recherchera la valeur dans A2.
    8. Entrez une virgule après la plage nommée, puis tapez le nombre 1. Ceci définit l'index de colonne à renvoyer. Pour vous, cela signifie que si l'ID en A2 se trouve dans la plage nommée, l'ID sera placé dans la cellule.
    9. Entrez une virgule après le chiffre 1, puis entrez FALSE , puis tapez les parenthèses fermantes. Cette option indique au VLOOKUP() Si les colonnes sont triées et que la valeur exacte est introuvable, la valeur la plus proche doit être renvoyée. Puisque vous ne voulez pas que cela se produise, FALSE est la valeur correcte à saisir.
    10. Assurez-vous que votre fonction est correctement définie, comme illustré ici, et appuyez sur Entrer. La cellule B2 affichera l'ID de A2, ou #N/A.
    11. Cliquez à nouveau dans la cellule B2 et assurez-vous qu'elle est en surbrillance.
    12. Pour copier la formule dans toute la feuille de calcul, double-cliquez sur le carré bleu en bas à droite de la cellule en surbrillance, qui est actuellement B2. Dans la colonne B, vous verrez maintenant soit les ID répétés de la colonne A, soit #N/A. Si vous voyez l'ID, cela vous indique que la vidéo représentée par cet ID a été trouvée dans le rapport Analytics. En d'autres termes, il a des vues vidéo. Et comme vous pouvez le supposer, si vous voyez #N/A la vidéo représentée par l'ID dans la colonne A n'a pas de vues.

    Supprimer les vidéos sans vues

    Vous allez maintenant trier la feuille de calcul afin que tous les ID de vidéos sans vue soient regroupés. Vous copiez ensuite ces ID et le format pour la suppression de vidéo en masse.

    1. Mettez en surbrillance la colonne B, puis dans le menu de la feuille de calcul, sélectionnez Données> Trier la feuille par colonne B, Z -> A . Cela regroupe toutes les vidéos sans vues en haut de la feuille de calcul, représentées par #N/A dans la cellule.
    2. Vous avez maintenant deux options sur la façon de procéder. Il s'agit de :
      • Si vous avez un petit nombre de vidéos à supprimer, vous pouvez simplement aller dans Studio et supprimer manuellement dans le MÉDIAS module.
      • Si vous avez un nombre important de vidéos à supprimer, vous souhaiterez peut-être continuer à effectuer les étapes suivantes pour alléger le fardeau de la suppression des vidéos.
    3. Mettez en surbrillance, puis copiez UNIQUEMENT les ID vidéo des vidéos que vous souhaitez supprimer. N'incluez aucune vidéo sans identifiant ou nom, car cela signifie que la vidéo est prête à être supprimée.
    4. Ouvrez un document Google Doc.
    5. Dans le menu du document, sélectionnez Edition> Coller sans mise en forme. Vous verrez simplement les ID vidéo dans une colonne. Pour effacer les vidéos en bloc, elles doivent être au format identifiant: 12345678 identifiant: 23456789 .... Plutôt que de modifier la liste à la main, continuez avec ces étapes.
    6. Sélectionner Edition> Rechercher et remplacer ...
    7. Pour le Trouver valeur entrer \n. Cela représente un retour chariot.
    8. Pour le Remplacer par valeur entrez un espace suivi de id:. L'espace devant le id: est cruciale pour un formatage correct.
    9. Vérifier la Correspondance à l'aide d'expressions régulières case à cocher.
    10. Clique le Remplace tout bouton, puis fermez la boîte de dialogue.
    11. Assurez-vous que votre liste d'identifiants vidéo s'affiche désormais de la même manière que: 
        1754261702001 id:4082515861001 id:5185914204001 id:5324448023001
  id:1754261492001 id:2180442442001 id:2535301903001 id:4713495235001
  id:4874718207001 id:4997278349001 id:5686632029001 id:5998203290001
  id:5998205402001 id:5998658922001 id:5998662608001 id:
    12. Pour terminer le formatage, vous devrez placer un id: devant le premier ID vidéo. Retirez ensuite le id: à la fin de la liste qui n'a pas d'ID vidéo réel correspondant.
    13. Copiez la liste correctement formatée des ID vidéo. 
        id:1754261702001 id:4082515861001 id:5185914204001 id:5324448023001
  id:1754261492001 id:2180442442001 id:2535301903001 id:4713495235001
  id:4874718207001 id:4997278349001 id:5686632029001 id:5998203290001
  id:5998205402001 id:5998658922001 id:5998662608001
    14. Ouvrez Studio MÉDIAS module.
    15. Collez votre liste d'identifiants vidéo dans le champ de recherche:
      Si vous avez un très grand nombre de vidéos à supprimer, il sera préférable de prendre plusieurs ensembles d'identifiants plutôt que tous les identifiants à la fois. Lors des tests, 100 vidéos à la fois ont fonctionné avec succès.
    16. presse Entrer.
    17. Vous verrez maintenant toutes les vidéos répertoriées qui correspondent aux identifiants que vous avez sélectionnés précédemment. Vous pouvez désormais sélectionner individuellement les vidéos à supprimer ou cocher la case Sélectionner tout et supprimer les vidéos.
