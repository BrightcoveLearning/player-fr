Utilisation d'un lecteur d'écran avec le lecteur Brightcove

{{ page.description }}

Le lecteur d'écran permet aux utilisateurs aveugles et malvoyants d'écouter les actions d'une vidéo. Pour ce faire, il lit un fichier de descriptions WebVTT défini. Ceci est utile s'il y a peu ou pas de conversation dans votre vidéo.

Cet exemple utilise l'application VoiceOver d'Apple, qui est intégrée au système d'exploitation Mac OS X. Pour plus de détails, consultez le Prise en main de VoiceOver document.

Pour obtenir la liste des lecteurs d'écran compatibles avec Brightcove Player, consultez le Accessibilité du lecteur Brightcove document.

Cet exemple fonctionne mieux dans le navigateur Web Chrome. Le navigateur Safari ne prend actuellement pas en charge les pistes de texte de description. Vous pouvez rencontrer des problèmes avec d'autres navigateurs, comme Firefox, qui ne prend en charge que VoiceOver de base.

Le joueur

Cet exemple utilise un fichier WebVTT pour décrire ce qui se passe dans la vidéo. Le texte de description est lu par un lecteur d'écran afin que les utilisateurs malvoyants puissent suivre l'action dans la vidéo.

 

Utilisez un lecteur d'écran avec cette vidéo

Suivez les étapes ci-dessous pour activer les descriptions et le lecteur d'écran.

  1. Démarrez la lecture de la vidéo ci-dessus et arrêtez-la lorsque vous pouvez voir les commandes du lecteur.
  2. Dans la barre de contrôle, passez la souris sur le Descriptions icône et sélectionnez les descriptions en anglais, en description.
  3. Vous devriez maintenant voir le texte du fichier de descriptions, affiché dans le lecteur.
  4. Nous allons maintenant activer le lecteur d'écran VoiceOver fourni avec une machine MacOS. Sélectionner CMD + F5 pour activer VoiceOver.
  5. Lorsque la boîte de dialogue modale VoiceOver apparaît, sélectionnez Utiliser VoiceOver bouton.
  6. Une barre de contrôle VoiceOver apparaîtra, dans laquelle vous verrez et entendrez le texte du fichier de description de la vidéo. Vous verrez et entendrez également toutes les actions que vous effectuez sur votre ordinateur.
  7. Sélectionner CMD + F5 pour désactiver VoiceOver.

Préparez votre vidéo

Pour utiliser l'icône de description dans la barre de contrôle du lecteur, vous devez créer un fichier de piste texte WebVTT et l'associer à votre vidéo. Pour savoir comment procéder, procédez comme suit:

  1. Créez un fichier de suivi de texte décrivant les actions de votre vidéo. Ce fichier revient à créer un fichier de sous-titres, sauf que vous l'appliquerez à votre vidéo en tant que type de description lors d'une étape ultérieure. Pour plus de détails sur la création de ce fichier, consultez le Ajout de légendes aux vidéos à l'aide du module multimédia document.

    Votre fichier de suivi de texte doit ressembler à ceci:

    WEBVTT

00:00:00.000 --> 00:00:05.000
[Herons...]

00:00:05.000 --> 00:00:10.000
[Great Blue ones...]

00:00:10.000 --> 00:00:15.000
[Nesting...]

00:00:15.000 --> 00:00:20.000
[Enjoy...]

00:00:20.000 --> 00:00:25.000
[Florida 2016]

  2. Les clients Brightcove Player peuvent apprendre à Ajouter des légendes aux vidéos par programmation.

    Ensuite, vous associez votre fichier de piste texte à votre vidéo. Les clients de Video Cloud peuvent se connecter à Studio et accéder au Médias module.

  1. Trouvez le Pistes de texte et sélectionnez la Éditer bouton.
  2. Téléchargez votre fichier de suivi de texte ou spécifiez l'URL externe.
  3. Entrer le code de langue à deux lettres pour la langue des données de piste de texte. Dans cet exemple, vous entrerez fr pour l'anglais. Vérifiez les langues prises en charge par votre lecteur d'écran.
  4. Met le Gentil de la piste de texte à descriptions.
  5. Laissez la valeur par défaut Type MIME valeur. Vous pouvez sélectionner défaut pour définir cette langue comme langue par défaut.
  6. Sélectionnez le Ajouter une piste de texte pour enregistrer les informations de piste de texte.
  1. Vous êtes maintenant prêt à utiliser cette vidéo avec un lecteur Brightcove et un lecteur d'écran.
