Versions de la documentation de Brightcove Player

{{ page.description }}

La documentation de Brightcove Player contient deux versions d'informations :

  • Utilisation du lecteur Brightcove avec un compte Video Cloud
  • Utilisation du lecteur Brightcove en mode autonome (Perform) (avec du contenu provenant d'une autre source)
Notez que la grande majorité du contenu est identique pour les deux versions, vous ne verrez donc peut-être rien changer visiblement lorsque vous passerez à l'autre version.

Il s'agit du même lecteur dans les deux cas, mais si vous avez un compte Video Cloud, le lecteur peut extraire les emplacements des vidéos et des ressources directement à partir de Video Cloud, de sorte que vous n'avez pas à fournir ces informations manuellement.

Lorsque cela fait une différence, nous avons créé des versions alternatives du contenu pour chaque scénario, et vous devez choisir la version selon laquelle le scénario correspond à votre situation.

Vous pouvez basculer entre les versions Video Cloud et Standalone Player du document à l'aide des boutons qui apparaissent en haut à droite de la page.
