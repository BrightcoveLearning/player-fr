Remarque : Comme mentionné dans la documentation du plugin FreeWheel, dans le Utilisation de l'URL correcte du SDK section, FreeWheel a supprimé l'URL dynamique du SDK au profit d'une URL versionnée. Cela signifie que le plugin Brightcove Player est désormais verrouillé sur une version spécifique par défaut, plutôt que sur la dernière version. Brightcove tentera de suivre les versions du SDK FreeWheel, mais peut être à la traîne.

Vous pouvez gérer votre propre version SDK dans leur intégration Brightcove Player. Cela implique de passer l' Html5.sdkurl option au plugin FreeWheel :

player.FreeWheelPlugin({
  Html5: {
    sdkurl: 'preferred-sdk-url'
  }
});
