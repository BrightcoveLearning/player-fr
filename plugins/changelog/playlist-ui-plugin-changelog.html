Journal des modifications du plug-in de l'interface utilisateur de la liste de lecture

Les modifications apportées au plugin sont les suivantes :

Changelog

7 mai 2020

v3.8.0

  • Option activée pour afficher les descriptions vidéo dans les miniatures de la liste de lecture

10 févr. 2020

v3.7.0

  • Ajout de traductions arabes
  • Correction d'une erreur renvoyée lors de l'initialisation lors de l'utilisation de la prochaine superposition

9 oct. 2019

v3.6.3

  • Afficher l'onglet Liste de lecture lorsque vous êtes en focus

8 oct. 2019

v3.6.2

  • Résolution d'un problème d'interface utilisateur de suite avec les playlists horizontales dans Firefox et IE11

9 sept. 2019

v3.6.1

  • Correction d'un problème avec les listes de lecture horizontales qui ne s'affichaient pas correctement sur IE11 et Firefox

27 août 2019

v3.6.0

  • Ajout de traductions pour le chinois, l'allemand, l'espagnol, le français, le japonais et le coréen

16 juil. 2019

v3.5.0

  • Ajoute la prise en charge de l'interface IFrame-like playlist pour les lecteurs dans la page via un élément conteneur

20 mars 2019

v3.4.4

  • Mettre à jour videojs-playlist à 4.3.1; cela résout les régressions introduites dans videojs-playlist 4.3.0

14 janv. 2019

v3.4.3

  • Mis à jour videojs-playlist à 4,3,0; cela résout un problème rare avec les listes de lecture contenant plusieurs éléments avec la même URL source

29 oct. 2018

v3.4.2

  • Ajoutée playlistPicker option, par défaut à true , pour permettre la suppression du sélecteur de playlist (lorsqu'il est passé comme false)
  • Correction de bug : Faire de la liste de lecture horizontale dans les joueurs iframe une hauteur statique de 20%, au lieu d'une hauteur variable
  • Correction de bug : Correction d'un problème où une barre de défilement verticale apparaissait sur les listes de lecture horizontales
  • Correction de bug : Supprimer l'interface utilisateur de la liste de lecture lorsque le lecteur est éliminé

3 oct. 2018

v3.4.1

  • Correction d'un problème où une barre de défilement verticale apparaissait sur les listes de lecture horizontales
  • Supprimer l'interface utilisateur de la liste de lecture lorsque le lecteur est éliminé

25 sept. 2018

v3.4.0

  • Ajout de l'option PlayListPicker, par défaut à true, pour permettre la suppression du sélecteur de liste de lecture.

17 sept. 2018

v3.3.2

  • Correction d'un problème dans lequel un lecteur de lecture chargé au mauvais moment pouvait empêcher le rendu de l'interface utilisateur
  • Suppression du script postinstall pour éviter les problèmes d'installation
  • Silencieux un play promettre une exception dans les listes de lecture
  • Mise à jour à l'aide du générateur de greffons

13 juin 2018

v3.3.1

  • Correction d'un problème qui faisait que le bouton de l'élément suivant n'avait pas de curseur

v3.2.1

v3.2.0

  • Ajout d'une option de lecture aléatoire pour randomiser la liste de lecture chaque fois qu'elle est modifiée
