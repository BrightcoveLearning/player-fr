    Plugin DVRUX Live

    S'il vous plaît voir l'avis que ce plugin est obsolète.
    Dans les versions de Brightcove Player antérieures à 6.33.0, le Plugin DVRUX en direct était nécessaire pour lire des flux en direct. Ce n'est plus le cas, car cette fonctionnalité est désormais intégrée dans Brightcove Player. Voir le Lecture de flux en direct document pour plus de détails. Le plugin Live DVRUX est désormais obsolète et ne devrait plus être utilisé.
