Plugins Brightcove

Voici un résumé des plugins de lecteur fournis par Brightcove. Les plugins sont utilisés pour améliorer Brightcove Player.

Notez que lorsque vous ajoutez un plugin dans le module Players, le menu déroulant du Brightcove plugins n'inclut pas tous les plugins listés ci-dessous. Si le plugin que vous souhaitez ajouter n'apparaît pas dans la liste, rafraîchissez la page et utilisez le Personnalisé option de plugin pour l'ajouter.
Plugin Chargé par défaut Description
Plugin vidéo 360º   Ce plugin transforme un élément vidéo correctement enregistré en une vidéo panoramique HTML5 à 360º qui peut être jouée dans Brightcove Player.
Plugin publicitaire uniquement   Ce plugin vous permet de lire uniquement des annonces IMA3 dans un lecteur. Aucun contenu vidéo normal n'est fourni.
Publicité avec FreeWheel   Ce plugin permet d'activer la technologie publicitaire FreeWheel, pour Flash ou HTML5, dans Brightcove Player.
Publicité avec IMA3   Intègre le lecteur Brightcove avec les publicités interactives Media Ads (IMA) de Google pour HTML5 version 3. Cela vous permet de demander et de suivre des annonces VAST pour votre lecteur.
Publicité avec SSAI   L'insertion publicitaire côté serveur (SSAI) vous permet d'intégrer des publicités dans vos vidéos afin qu'elles ne puissent pas être bloquées par les bloqueurs de publicités dans le navigateur. Par défaut, le plugin fait en sorte que toutes les publicités soient regardées et affiche un compte à rebours des annonces pendant qu'elles sont diffusées. Vous pouvez facilement personnaliser ce plugin pour ignorer les publicités.
Écran de fin personnalisé   Affiche le code HTML fourni par l'éditeur à la fin de la lecture vidéo.
Plugin Chromecast   Permet à Brightcove Player de diffuser des vidéos depuis votre bureau ou votre navigateur Android Chrome vers l'appareil Chromecast.
DRM (officiellement appelé DASH)   Utilisé pour implémenter la DRM dans Brightcove Player.
Afficher les aperçus miniatures   Affiche une image d'aperçu sur la chronologie à des points définis de la vidéo. Au fur et à mesure que l'utilisateur se déplace sur la chronologie, des images miniatures s'affichent pour des périodes spécifiées.
Messages d'erreur O Permet au lecteur d'afficher des messages conviviaux lorsqu'il rencontre une erreur. L'écran est une superposition semi-transparente et coiffée par la feuille de style par défaut.
Gestionnaire de balises Google   Permet l'intégration avec la fonctionnalité Google Tag Manager avec Brightcove Player.
HLS O Lit des vidéos HLS sur des plates-formes qui ne prennent pas en charge HLS mais qui possèdent Flash Player. L'option permet de lire le contenu vidéo du manifeste m3u8 dans le lecteur.
Affichage de superposition   Affiche des messages simples sous forme de superpositions semi-transparentes au-dessus du lecteur. L'affichage des messages peut être basé sur les événements du lecteur, tels que la lecture ou la pause, ou à une heure spécifiée pendant la lecture vidéo.
Plugin d'image dans l'image   Permet à l'utilisateur de faire défiler la page vers le bas et le lecteur sera épinglé à un coin de la page Web.
Plug-in d'interface utilisateur   Améliore l'apparence et la convivialité de la fonctionnalité de liste de lecture par défaut.
Sélection de la qualité   Fournit un bouton de menu dans la barre de contrôle du lecteur qui permet de sélectionner manuellement la qualité de lecture pour les sources HLS ou Dash.
Médias sociaux   Permet aux utilisateurs de partager une vidéo sur Facebook, Twitter, Tumblr, Pinterest ou LinkedIn.
Tealium   Permet l'intégration avec le gestionnaire de balises Tealium iQ.
Recherche de vignettes   Active les aperçus miniatures à afficher lors du survol de la barre de progression.

Plugins Open Source

Le lecteur Video.js open source, sur lequel repose Brightcove Player, dispose de nombreux plugins conçus pour lui. Ces plugins sont compatibles avec Brightcove Player tant que la version du lecteur Video.js est la même version sur laquelle repose le Brightcove Player. Le lecteur Brightcove peut être en version avant ou derrière le lecteur Video.js open source. Le lien vers la page Plugins Video.js est https://github.com/videojs/video.js/wiki/Plugins.

Des plugins open source spécifiques à Brightcove Player sont également disponibles. A titre d'exemple, les plugins pour Conviva et Nielsen analytics. Notez que Brightcove ne prend pas en charge ou ne documente pas ces plugins. Les clients doivent contacter leur responsable de compte chez Conviva/Nielsen pour obtenir plus de détails et y accéder.

Disponible auprès de Moat, une société qui fournit aux éditeurs un moyen simple de mesurer et de rapporter la visibilité des annonces diffusées, est un plugin qui fonctionne avec Brightcove Player. Pour accéder au plugin de Moat pour Brightcove Player, les clients doivent contacter leur responsable de compte Moat.
