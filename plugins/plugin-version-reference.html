Utiliser en HTML

Si vous utilisez les URL des plugins directement dans une page HTML, vous utiliseriez un <script> balise pour le JavaScript et un <link> balise pour le CSS. Des exemples suivent:

<script src="https://players.brightcove.net/videojs-vr/1/videojs-vr.min.js"></script>
<link href="https://players.brightcove.net/videojs-vr/1/videojs-vr.css" rel="stylesheet">
