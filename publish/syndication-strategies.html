Stratégies de syndication

{{ page.description }}

Syndication ouverte et virale

Si vous souhaitez que vos vidéos soient distribuées le plus possible, vous pouvez opter pour une démarche de syndication ouverte. Vous pouvez publier des lecteurs ou créer des pages Web à partir desquelles tous les visiteurs pourront obtenir le code de publication du lecteur et incruster le lecteur dans leur site Web.

La syndication virale est une sous-catégorie de la syndication ouverte. Le lecteur Brightcove offre des fonctionnalités qui permettent aux spectateurs de cliquer sur le lecteur et de partager la vidéo sur Facebook, Twitter ou d'autres sites de partage social, ou d'obtenir un code intégré du lecteur ou un lien URL vers le lecteur, ce qui leur permet de présenter votre lecteur et le contenu vidéo sur n'importe quel site Web ou par e-mail. En savoir plus sur Configuration des propriétés sociales d'un joueur.

La syndication virale peut être un outil très efficace pour élargir votre public. Néanmoins, vous devez céder un peu de votre contrôle en contrepartie. La syndication ouverte a l'inconvénient de retirer votre vidéo du contenu que vous avez créé pour elle sur votre site Web ou vos affiliés.' sites Internet. Vous pouvez prendre une mesure pour vous assurer que votre image de marque accompagne la vidéo - voir Champs de filigrane dans les profils d'ingestion.

Syndication avec partenaires tiers de confiance

Dans la syndication avec partenaires tiers de confiance (également nommée syndication gérée), vous développez une relation contractuelle avec vos partenaires. Cette relation s’effectue souvent dans le cadre d’accords spécifiques liés au contenu qui sera rendu disponible et peut également inclure le paiement pour le contenu, ou bien un accord de partage des revenus publicitaires. Si la publicité fait partie de votre stratégie d'entreprise vidéo, assurez-vous d'avoir un accord clair avec vos partenaires de syndication quant à la réservation des droits de publicité en relation avec les vidéos syndiquées. En utilisant la syndication avec partenaires tiers de confiance, vous contrôlez qui peut diffuser votre contenu et l’endroit où il peut être publié.

Joueurs pour la syndication des partis de confiance

Si vous adoptez une approche de syndication de tiers de confiance, nous vous recommandons de créer un lecteur Brightcove dédié pour chacun de vos partenaires de syndication. Grâce à cette stratégie, vous pouvez enregistrer la performance de chacun des lecteurs, et cibler le contenu et les campagnes publicitaires selon le lecteur. Vous conservez également le contrôle sur le contenu publié sur le lecteur et pouvez mettre à jour ou supprimer du contenu à tout moment depuis le Studio. Voici les principales étapes de la mise en œuvre de cette stratégie :

  1. Dans le compte Brightcove, créez un lecteur séparé pour chaque partenaire de syndication. Vous pouvez Personnaliser tout ce que vous voulez, avec la contribution du partenaire.
    Adoptez une convention de nommage qui vous permettra d'identifier facilement quel affilié correspond à quel joueur, et qui vous permettra de séparer les joueurs syndiqués des joueurs de votre propre site.
  2. Utilisez la restriction de domaine pour restreindre le lecteur au site Web du partenaire. Ceci garantit que les internautes ne pourront pas se procurer le code incrusté à partir du site du partenaire et le publier sur un autre site. Vérifiez les paramètres de l’autre lecteur. Vous voudrez certainement configurer les options d’hébergement du lecteur pour que les liens guident le trafic vers votre site ou pour désactiver la publicité pour le lecteur. Pensez aux paramètres de lecteur fournis par Video Cloud et configurez le lecteur conformément à l’accord que vous avez conclu avec votre partenaire de syndication.
  3. Choisissez le contenu pour le joueur. S’il s’agit d’un lecteur de sélection, vous contrôlez quelles vidéos sont, de temps en temps, visibles dans chacun des lecteurs des partenaires. Ou bien, s’il s’agit d’un lecteur de vidéo unique, vous pouvez le programmer grâce à la publication rapide de vidéo ou en modifiant le code de publication du lecteur.
  4. Distribuez le code de publication du lecteur. Vous pouvez envoyer le code joueur à vos partenaires de syndication, qui peuvent ensuite intégrer leur joueur spécifique dans leur propre page web. Vous pouvez utiliser le code d'intégration Standard (iframe) ou Avancé (balise vidéo). Mais la distribution du code intégré Standard vous permet de maintenir un contrôle plus strict du joueur, car l'affilié ne pourra pas modifier le joueur en dehors de sa taille.
  5. Surveiller les données de performance. Vous souhaitez connaître la performance de vos vidéos dans chacun des sites de vos partenaires. Ceci s’avère particulièrement important si vous diffusez des publicités ou partagez les revenus de votre contenu syndiqué. Puisque vous avez créé un joueur spécifiquement pour chaque affilié, il est facile d'examiner les performances par affilié dans Video Cloud Analytics.
