Référence des erreurs du lecteur Brightcove

Ce document est la collection d'erreurs pour Brightcove Player. Notez que les erreurs sont générés à partir d'un certain nombre de plugins différents.

Types d'erreurs

Il y a six principaux types d'erreurs qui sont brièvement décrits dans cette section. Vous pouvez cliquer sur le type d'erreur pour voir les erreurs spécifiques à chaque type, le cas échéant.

  • Erreurs standard envoyées depuis le plugin Errors: Ces noms d'erreur commencent par MEDIA_ERR.
  • Erreurs personnalisées définies par Brightcove et envoyées à partir du plugin Errors: Ces noms d'erreur commencent par PLAYER_ERR.
  • Erreurs envoyées à partir du catalogue Video Cloud: Ces noms d'erreur commencent par VIDEO_CLOUD_ERR. Bien sûr, Lecteur Brightcove les clients ne verront jamais ces erreurs.
  • Erreurs envoyées depuis le plugin HLS: Toutes ces erreurs ont le même type, APPEND_BUFFER_ERR et le message est récupéré à partir de l'erreur native du navigateur.
  • Erreurs signalées dans la console des outils de développement du navigateur: Ces erreurs sont orientées développeur car elles ne seront observées que dans la console d'un navigateur.
  • Erreurs personnalisées définies par l'utilisateur et envoyées par le plug-in d'erreurs: Ces noms d'erreur sont définis par l'utilisateur, par exemple AGE_GATE_ERROR.
  • Erreurs de sécurité réseau

Erreurs types

Très généralement, des erreurs de média se produisent lorsque des problèmes surviennent en essayant de récupérer ou de lire le média. Par exemple, si votre connexion réseau est perdue ou si vous obtenez une vidéo cryptée qui ne peut pas être déchiffrée, vous obtiendrez une erreur multimédia. Pour plus de détails, consultez le Erreurs standard section de la Afficher le plugin des messages d'erreur document.

Code Type Manchettes Message
1 MEDIA_ERR_ABORTED Le téléchargement de la vidéo a été annulé Vous avez abandonné la lecture multimédia.
2 MEDIA_ERR_NETWORK La connexion vidéo a été perdue, veuillez confirmer que vous êtes connecté à Internet Une erreur réseau a provoqué un échec partiel du téléchargement du média. Actuellement plus utile pour les formats vidéo MP4 et/ou téléchargement progressif. Voir le Problèmes connus section de la Afficher le plugin des messages d'erreur document pour plus de détails.
3 MEDIA_ERR_DECODE La vidéo est mauvaise ou dans un format qui ne peut pas être lu sur votre navigateur La lecture multimédia a été interrompue en raison d'un problème de corruption ou parce que les médias utilisés ne sont pas pris en charge par votre navigateur.
4 MEDIA_ERR_SRC_NOT_SUPPORTED Cette vidéo est indisponible ou non prise en charge par ce navigateur. Le support n'a pas pu être chargé, soit parce que le serveur ou le réseau a échoué ou parce que le format n'est pas pris en charge.
5 MEDIA_ERR_ENCRYPTED La vidéo que vous essayez de regarder est cryptée et nous ne savons pas comment la déchiffrer La vidéo que vous essayez de regarder est cryptée et nous ne savons pas comment la déchiffrer.
Inconnu MEDIA_ERR_UNKNOWN S. O. Un problème imprévu a été rencontré, revenez bientôt et réessayez.

Erreurs personnalisées

Très généralement, des erreurs de lecteur se produisent lorsque la vidéo a été chargée, pour une raison quelconque, ne peut pas être lue. Par exemple, si des restrictions de domaine, de géo ou d'IP empêchent l'affichage d'une vidéo, des erreurs de lecteur se produiront. Pour plus de détails, consultez le Erreurs personnalisées section de la Afficher le plugin des messages d'erreur document.

Code Message Description
-1 PLAYER_ERR_NO_SRC Aucune vidéo n'a été chargée.
-2 PLAYER_ERR_TIMEOUT Impossible de télécharger la vidéo, le délai d'expiration par défaut est de 45 secondes.
Non réglé PLAYER_ERR_DOMAIN_RESTRICTED Cette vidéo est restreinte à la lecture sur votre domaine actuel.
Non défini PLAYER_ERR_IP_RESTRICTED Cette vidéo est restreinte à votre adresse IP actuelle.
Non défini PLAYER_ERR_GEO_RESTRICTED Cette vidéo n'est pas diffusée dans votre région géographique actuelle.

Erreurs de catalogue

Très généralement, le catalogue génère des erreurs lorsqu'il y a des problèmes lors de la récupération ou de la lecture initiale de la vidéo ou de la playlist. Par exemple, si vous utilisez un ID inexistant, ou un ID d'un autre compte dans lequel le lecteur existe, pour récupérer une vidéo, vous obtiendrez une erreur de catalogue. Pour plus de détails, consultez le Erreurs de catalogue section de la Catalogue des joueurs document.

Erreur de catalogue Description
VIDEO_CLOUD_ERR_ACCOUNT_NOT_FOUND Le lecteur a été configuré avec un ID de compte Video Cloud non valide.
VIDEO_CLOUD_ERR_RESOURCE_NOT_FOUND L'ID de la liste de lecture par défaut n'est pas valide.
VIDEO_CLOUD_ERR_VIDEO_NOT_FOUND La vidéo par défaut n'est pas valide.
VIDEO_CLOUD_ERR_NOT_PLAYABLE La vidéo Video Cloud n'est pas lisible.
Les causes fréquentes de cette erreur incluent la désactivation d'un ID vidéo ou l'expiration de la disponibilité programmée.

Erreurs du plugin HLS

Pour plus de détails, consultez le les erreurs section de la Plug-in HLS document.

Erreur du plugin HLS Description
APPEND_BUFFER_ERR Le navigateur a lancé une erreur lors de la lecture d'une vidéo HLS.

Erreurs de console

Ces erreurs apparaîtraient dans une console ressemblant à ceci :

Message de la console Solution
VIDEOJS: ERROR: videojs-contrib-ads n'a pas vu d'événement loadstart 5 secondes après avoir été initialisé, mais une source est présente. Cela indique que videojs-contrib-ads a été initialisé trop tard. Il doit être initialisé immédiatement après video.js dans la même coche. Par conséquent, certaines annonces ne seront pas diffusées et certains événements multimédias seront incorrects. Pour plus d'informations, voir https://github.com/videojs/videojs-contrib-ads#important-note-about-initialization Vous voyez cela parce que pour que le plugin IMA3 fonctionne correctement, il doit être chargé juste après l'initialisation du joueur, avant que d'autres plugins ne soient initialisés. Voir le Implémenter en utilisant du code section de la Plug-in IMA3 doc pour la solution.

Erreurs utilisateur personnalisées

Pour plus de détails, consultez le Ajout d'erreurs personnalisées section de la Afficher le plugin des messages d'erreur document. Il n'y a pas d'erreurs utilisateur personnalisées standard.

Erreurs de sécurité réseau

Ces erreurs se produisent lorsque la sécurité réseau bloque certains domaines et ports Brightcove.

Code d'erreur Description
FLASHLS_ERR_CROSS_DOMAIN Impossible de charger la vidéo : accès interdomaine refusé.
Vous aurez peut-être besoin d'un crossdomain.xml à la racine de votre fichier de liste de lecture qui autorise le domaine à partir duquel vous demandez (ou '*').
