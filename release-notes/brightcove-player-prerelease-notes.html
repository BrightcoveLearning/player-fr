Notez que toutes les 1 à 2 semaines, les versions pré-publiées deviennent la version actuelle. À ce moment-là, tous les joueurs non verrouillés sont mis à niveau vers la version actuelle. Si vous souhaitez verrouiller votre lecteur sur une certaine version pour une raison quelconque, consultez le désactiver les mises à jour automatiques des lecteurs (site externe) .
 
 

Essayez une nouvelle version

Si une version préliminaire est disponible, le moyen le plus simple d'essayer une version préliminaire est d'aller sur Studio. JOUEURS module, cliquez sur le lecteur que vous souhaitez modifier, puis modifiez le Informations sur le joueur . Utilisez le sélecteur de version pour choisir la dernière pré-version:

Vous pouvez mettre à jour un lecteur vers la version préliminaire à l'aide d'une instruction curl. Un aperçu rapide de la boucle peut être trouvé en haut de la Pas à pas: Gestion des joueurs (site externe). Dans le document, il y a une section spécifique sur la façon de mettre à jour votre lecteur.

Voici la commande curl pour mettre à jour votre lecteur, où vous substituez les valeurs appropriées pour VOTRE_EMAIL , VOTRE_ACCOUNT_ID et VOTRE_PLAYER_ID . 

curl \
  --header "Content-Type: application/json" \
  --user YOUR_EMAIL \
  --request PATCH \
  --data '{
    "player": {
    "template": {
      "version": "6.57.0"
      }
    }
  }' \
  https://players.api.brightcove.com/v2/accounts/YOUR_ACCOUNT_ID/players/YOUR_PLAYER_ID/configuration

Dans la plupart des cas, vous souhaiterez publier le lecteur, ce qui peut être fait avec l'instruction curl suivante. Encore une fois, vous devrez remplacer les valeurs correctes là où cela est indiqué. 

curl \
  --header "Content-Type: application/json" \
  --user YOUR_EMAIL  \
  --request POST \
  https://players.api.brightcove.com/v2/accounts/YOUR_ACCOUNT/players/YOUR_PLAYER_ID/publish
