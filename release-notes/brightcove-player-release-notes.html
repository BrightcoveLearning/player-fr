Remarque : Par défaut, les mises à jour sont appliquées automatiquement à tous les lecteurs. Pour empêcher la mise à jour automatique d'un lecteur, vous devez le définir sur la mise à jour manuelle (également connue sous le nom de verrouillage d'un lecteur). Vous pouvez le faire dans le module Studio Players (site externe) ou à l'aide de l'API Player Management.

La désactivation des mises à jour automatiques doit être une mesure temporaire, appliquée pendant que vous résolvez tous les problèmes créés par la mise à jour. Si vous ne mettez pas à jour les lecteurs vers la version actuelle, vous manquez les nouvelles fonctionnalités et corrections de bugs, et les anciennes versions du lecteur cesseront d'être prises en charge à un moment donné.

Pour les notes de version des anciens joueurs, consultez l'archive.

Pour les anciennes notes de version du plugin, consultez pages du plugin.
