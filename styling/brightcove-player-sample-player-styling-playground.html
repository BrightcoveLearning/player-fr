Exemple de lecteur Brightcove : Playground Styling Player

Cette rubrique vous offre un terrain de jeu pour expérimenter avec le style des joueurs.

Aire de jeux de style

Dans ce sujet, vous utiliserez un Codepen, illustré ci-dessous, comme une aire de jeux de style. Visualisez-le sur le codepen.io site pour apporter des modifications, et n'hésitez pas à charger le Codepen pour votre propre usage. Notez que vous voudrez peut-être masquer le panneau JavaScript dans Codepen, car il n'est utilisé pour rien ici, et agrandir le volet CSS.

Voir le stylo Brightcove Player Styling Playground par Brightcove Learning Services (@bcls ) sur CodePen.
