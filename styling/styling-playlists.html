Listes de lecture de style

Dans cette rubrique, vous apprendrez à styliser des listes de lecture.

Présentation

Si vous souhaitez modifier l'apparence des listes de lecture, vous pouvez le faire en utilisant CSS. Dans ce document, vous verrez un certain nombre de styles que vous pouvez modifier. Bien entendu, les changements possibles sont nombreux et vous devez trouver le bon sélecteur pour accomplir votre tâche.

Les documents suivants sont des informations essentielles pour compléter le tableau de l'utilisation des listes de lecture :
  • Implémentation de listes de lecture: Ce document montre les décisions que vous devez prendre concernant l'utilisation des listes de lecture et comment obtenir le code de joueur dont vous avez besoin pour les obtenir sur votre page HTML.
  • Plugin d'interface utilisateur de la playlist Ce document détaille les options disponibles pour améliorer l'apparence et la convivialité de la fonctionnalité de playlist.

Styliser une liste de lecture verticale

Le diagramme suivant résume les styles modifiés pour une liste de lecture verticale. Le diagramme suit un tableau qui répertorie les styles.

Contrôle de la couleur du texte de la liste de lecture .vjs-mouse.vjs-playlist {
  couleur : #E9D2B3 ;
}
Contrôlez la couleur de la
espaces entre les éléments de la liste de lecture		 .vjs-mouse.vjs-playlist {
  couleur d'arrière-plan : #161D6E ;
}
Contrôle de la couleur du texte de durée .vjs-playlist .vjs-playlist-duration {
  couleur : #9BB4DA ;
}
Contrôle de la couleur de l'arrière-plan de la durée .vjs-playlist .vjs-playlist-duration {
  couleur d'arrière-plan : #1160AA ;
}
Espace de contrôle entre les éléments de la liste de lecture .vjs-playlist-vertical .vjs-playlist-item {
  marge supérieure : 20px ;
}
Pour éviter une marge au-dessus du premier élément de la liste de lecture, vous devez suivre le style de réglage de la marge comme suit : 
.vjs-playlist-vertical .vjs-playlist-item:first-child {
  margin-top: 0px;
}

Masquer la barre de défilement

Une barre de défilement sur la liste de lecture horizontale n'est pas nécessaire, mais apparaîtra. Si vous souhaitez masquer la barre de défilement, le style suivant fait exactement cela:

div.vjs-playlist.vjs-playlist-horizontal {
  overflow-y: hidden;
}
