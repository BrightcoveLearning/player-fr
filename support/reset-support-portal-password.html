Introduction

Si vous oubliez votre mot de passe pour le portail de support Brightcove ou si vous souhaitez simplement le modifier, c'est très facile à faire. Les étapes ci-dessous vous guideront tout au long du processus.

Notez que les informations d'identification du portail de support ne sont pas associées aux informations d'identification Studio.

Étapes

  1. Accédez au portail de support.
  2. Cliquez sur le bouton « Mot de passe oublié ? « lien :
  3. Dans le Mot de passe oublié dialogue, entrez votre nom d'utilisateur et cliquez sur Continuer:
  4. Dans quelques minutes, vous devriez recevoir un e-mail contenant un lien vers la page de réinitialisation du mot de passe :
  5. Cliquez sur le lien, puis dans la page qui s'ouvre, entrez et confirmez votre nouveau mot de passe, puis cliquez sur Modifier le mot de passe:
  6. Votre mot de passe sera immédiatement mis à jour et vous serez redirigé vers le portail d'assistance.
