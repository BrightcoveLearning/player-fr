Informations sur le lecteur Brightcove Modal

{{ page.description }}

Afficher les informations du joueur

Vous affichez les informations du lecteur en cliquant avec le bouton droit sur le lecteur et en sélectionnant Informations sur le joueur , comme le montre cette capture d'écran.

Remarque : Sur mobile, vous pouvez utiliser la pression longue pour ouvrir le modal.

Une fois que vous sélectionnez Informations sur le joueur vous verrez la fenêtre modale contenant des informations sur le joueur, comme indiqué dans cette capture d'écran.

Remarque : Seuls les plugins fournis par Brightcove qui font partie de la configuration apparaîtront dans le Plugins section. Si vous ajoutez un plugin Brightcove dans le JavaScript d'une page HTML, il n'apparaîtra PAS dans la liste. Les plugins développés par le client, même s'ils sont répertoriés dans la configuration du lecteur, n'apparaîtront PAS.

Fermer la fenêtre modale

Vous fermez le modal soit en appuyant sur ESC ou en cliquant sur l'icône de fermeture en haut à gauche.

Modifier le menu contextuel

Vous pouvez modifier le menu qui apparaît lors du clic droit, appelé menu contextuel. Pour ce faire, vous devez modifier la configuration de votre lecteur. Vous pouvez le faire de deux manières:

  • Utilisation de Studio Éditeur JSON du JOUEURS module.
  • Un outil pour effectuer le changement de configuration est boucle.

Les deux modifications que vous pouvez apporter et la manière de les affecter sont présentées ici.

Remarque : Les instructions curl ci-dessous utilisent l'API de gestion du lecteur pour modifier la configuration du lecteur. Dans les instructions curl, vous verrez des valeurs de $EMAIL, $ACCOUNT_ID et $PLAYER_ID. Vous devez soit créer des variables d'environnement par ces noms et affecter les valeurs correspondantes, soit fournir les valeurs littérales des variables.

Empêcher le menu d'apparaître

Si vous souhaitez empêcher le menu contextuel des informations sur le lecteur d'apparaître, vous pouvez effectuer l'une des opérations suivantes:

Utiliser l'éditeur JSON

Modifiez la configuration du joueur en ajoutant la propriété mise en surbrillance (veillez à enregistrer la modification) :

Utiliser curl

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request PATCH \
  --data '{
      "context_menu": false
  }' \
  https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/configuration
Remarque : Chaque fois que vous modifiez un joueur à l'aide de curl, vous devez republier le lecteur. Vous pouvez le faire dans Studio, et vous verrez que votre lecteur est marqué comme devant être publié dans le Statut colonne:

Vous pouvez également utiliser curl pour publier votre lecteur :

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request POST \
    https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/publish

Empêcher l'option de menu Brightcove

Si vous souhaitez empêcher le À propos de Brightcove option de menu apparaissant, vous pouvez effectuer l'une des opérations suivantes:

Utiliser l'éditeur JSON

Modifiez la configuration du joueur en ajoutant la propriété mise en surbrillance (veillez à enregistrer la modification) :

Utiliser curl

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request PATCH \
  --data '{
    "context_menu": {
      "brightcove_branding": false
    }
  }' \
  https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/configuration
Remarque : Chaque fois que vous modifiez un joueur à l'aide de curl, vous devez republier le lecteur. Vous pouvez le faire dans Studio, et vous verrez que votre lecteur est marqué comme devant être publié dans le Statut colonne:

Vous pouvez également utiliser curl pour publier votre lecteur :

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request POST \
  https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/publish
