Navigateurs pris en charge

Nous vous recommandons fortement de vérifier régulièrement les mises à jour disponibles du navigateur afin de vous assurer que vous disposez des correctifs et performances les plus récents et les plus performants.

Pour une liste complète des navigateurs pris en charge, consultez le Configuration système requise pour Brightcove Player document.

Messages d'erreur courants

Voici une liste des messages d'erreur que vous pouvez recevoir lorsque vous tentez de lire des vidéos, ainsi que des étapes suggérées pour essayer de résoudre le problème :

MEDIA_ERR_ABORTED

  • Code d'erreur : 1
  • Headline : Le téléchargement de la vidéo a été annulé
  • Message : Vous avez abandonné la lecture multimédia

Cause possible

Une erreur de navigateur un peu générique qui indique que quelque chose a arrêté le processus de téléchargement vidéo dans ses pistes. Cela pourrait être :

  • Un problème avec l'ordinateur, tel qu'un plantage ou une interruption du service réseau
  • Un problème de navigateur ou de configuration
  • Le navigateur n'est pas pris en charge ou manque actuellement de support pour la vidéo
  • Un problème de pare-feu, de proxy Web ou de réseau bloque le téléchargement de la vidéo lorsque le navigateur tente de le lire

Étapes à suivre pour essayer

Voici quelques étapes pour tenter de résoudre le problème :

  • Actualiser la page
  • Essayez un autre navigateur

RÉSEAU MEDIA_ERR_

  • Code d'erreur : 2
  • Manchettes : La connexion vidéo a été perdue, veuillez confirmer que vous êtes connecté à Internet
  • Message : Une erreur réseau a provoqué un échec partiel du téléchargement du média

Cause possible

Voici quelques causes possibles :

  • Un pare-feu bloquant les données vidéo
  • Un proxy Web interrompt la connexion ou filtrage du trafic
  • Logiciel de filtrage (tel que Sophos ou Lightspeed Web Filter bloquant le contenu)
  • Une panne générale du réseau (par exemple, la connexion Internet à votre maison/école ou ordinateur personnel échoue)

Étapes à suivre pour essayer

Voici quelques étapes pour essayer de résoudre le problème :

  • Vérifiez que Internet fonctionne correctement sur la machine concernée
  • Actualisez la page, ce qui peut redémarrer une connexion qui a peut-être été interrompue
  • Essayez une autre vidéo. Est-ce que ça marche ? Si ce n'est pas le cas, cela suggère que le contenu dans son ensemble ne peut pas être atteint.

MEDIA_ERR_DECODE

  • Code d'erreur : 3
  • Manchettes : La vidéo est mauvaise ou dans un format qui ne peut pas être lu sur votre navigateur
  • Message : La lecture du média a été interrompue en raison d'un problème de corruption ou parce que les fonctionnalités de support utilisées par votre navigateur ne supportaient pas

Cause possible

Un navigateur ne pouvant pas lire le contenu demandé. Cela peut se produire pendant la lecture, ou plus probablement, au début ou vers le début.

Étapes à suivre pour essayer

Voici quelques étapes pour tenter de résoudre le problème :

  • Actualisez la page, ce qui peut télécharger à nouveau un morceau de contenu qui n'a pas pu être lu en raison d'un téléchargement partiel lors du premier essai.
  • Mettez à jour votre navigateur vers la dernière version. Chrome et Firefox devraient vous inviter avec des mises à jour, mais il est utile de vérifier.

MEDIA_ERR_SRC_NOT_SUPPORTÉ

  • Code d'erreur : 4
  • Manchettes : Cette vidéo est indisponible ou non prise en charge par ce navigateur.
  • Message : Impossible de charger le support, soit parce que le serveur ou le réseau a échoué ou parce que le format n'est pas pris en charge

Cause possible

Un navigateur ne pouvant pas lire le contenu demandé en raison de son indisponibilité, ou ne peut pas être lu dans ce navigateur particulier. Il diffère de l'erreur 3 en ce que la lecture échoue avant qu'elle ne commence, plutôt que pendant la lecture.

Étapes à suivre pour essayer

Dans la plupart des cas, ces deux paramètres sont corrects, mais peuvent avoir été spécifiquement définis par votre équipe informatique.

PLAYER_ERR_TIMEOUT

  • Code d'erreur: -2
  • Manchettes : Impossible de télécharger la vidéo.
  • Message : Impossible de télécharger la vidéo.

Le lecteur ne parvient pas à lire le contenu et les tampons pendant 45 secondes

Cause possible

  • Bande passant/vitesse réseau insuffisante
  • Un pare-feu bloquant les données vidéo
  • Un proxy Web interrompt la connexion ou filtrage du trafic
  • Logiciel de filtrage (tel que Sophos ou Lightspeed Web Filter bloquant le contenu)
  • Une panne générale du réseau (par exemple, la connexion Internet à votre maison/école ou ordinateur personnel échoue)
  • Parfois, il peut être causé par des bugs spécifiques au joueur

Étapes à suivre pour essayer

Voici quelques étapes pour essayer de résoudre le problème :

  • Vérifiez que Internet fonctionne correctement sur la machine concernée
  • Actualisez la page, ce qui peut redémarrer une connexion qui a peut-être été interrompue
  • Essayez une autre vidéo. Est-ce que ça marche ? Si ce n'est pas le cas, il suggère que le contenu dans son ensemble ne peut pas être atteint.
  • Le problème se produit-il toujours au même moment dans le temps ?

Débogueur Player

Le débogueur Brightcove Player est un plugin que vous pouvez facilement attacher à votre lecteur pour capturer et enregistrer des informations sur votre lecteur, vos médias et vos publicités.

Pour des étapes détaillées sur la configuration et l'utilisation du débogueur de lecteur, consultez le Pas à pas: Débogueur de lecteur Brightcove document.

Capturer des informations

Les informations suivantes vous aideront à étudier votre problème.

Informations sur l'utilisateur final

Recueillez les informations suivantes sur votre environnement système :

  • Prenez une capture d'écran ou exportez les détails de l'environnement depuis http://supportdetails.com/. Ces détails comprennent :
    • Système d'exploitation
    • Résolution d'écran/Taille du navigateur
    • Navigateur Web, y compris la version
    • l'adresse IP ;
  • Prenez une capture d'écran ou exportez les résultats de la vitesse de votre réseau à partir de http://beta.speedtest.net/

Informations utiles

Créez et enregistrez un journal réseau à partir de votre navigateur.

Les instructions ci-dessous concernent Chrome, mais les mêmes instructions s'appliquent à tous les navigateurs pris en charge
  • Ouvrez l'outil de développement Chrome. (F12 sur Windows/CMD+OPT+I sur Mac)
  • Sélectionnez le Console onglet pour déterminer si des erreurs sont signalées.
  • Sélectionnez le Réseau languette.
  • Faites un clic droit et sélectionnez Enregistrer sous HAR avec contenu.
  • Sauver la .har fichier (l'extension est .xml avec IE) sur votre ordinateur pour un examen plus approfondi.

Pour des instructions plus détaillées, consultez le Création d'un journal réseau document.

Contacter le support

Si vous n'avez pas résolu votre problème avec les étapes du message d'erreur ou le débogueur du lecteur, vous pouvez soumettre une demande d'assistance.

Informations essentielles de dépannage

Les informations suivantes sont essentielles pour résoudre votre problème :

  • URL ou vidéo Nom du contenu affecté
  • Un délai approximatif au cours de laquelle le problème est survenu ou a commencé, sinon en cours, que nous pouvons utiliser pour corréler les constatations/déploiements/problèmes systèmes/problèmes connus
  • Clarifier le type de problème (Ne joue pas, ne charge pas, pause et mise en mémoire tampon, mauvaise qualité, sauts ou coupes vidéo, pas de son etc...)
  • Une brève déclaration expliquant les étapes suivies pour reproduire la question si elle n'est pas immédiatement évidente
  • Indiquez la configuration où le problème se produit et s'il s'agit d'un ordinateur de bureau ou d'un mobile (plate-forme du système d'exploitation, navigateurs et versions)
  • Confirmer, dans la mesure du possible, si cela se produit pour plusieurs utilisateurs et si un ou plusieurs réseaux semblent être affectés
  • Fournir des captures d'écran ou le texte du message d'erreur selon le cas
  • Journaux HAR de la console du navigateur (voir les instructions dans la section précédente)
  • ID de session qui s'affiche dans le message d'erreur. Fournissez l'ID de session ou prenez plus facilement une capture d'écran du message d'erreur.

Soumettre votre problème

Utilisez les options suivantes pour saisir votre problème à l'équipe d'assistance :
