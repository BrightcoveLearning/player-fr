Propriétés du joueur

Les propriétés du lecteur Brightcove contiennent des informations sur votre lecteur spécifique. Pour plus de détails sur la configuration du skin et de la version du lecteur, consultez le Skins et versions de joueur document.

VERSION

le bc.VERSION property spécifie la version du lecteur telle que définie dans la configuration du lecteur.

Voici un exemple :

    <video-js id="video_1" style="width: 640px; height: 360px;"
    data-video-id="4029697544001"
    data-account="1752604059001"
    data-player="default"
    data-embed="default"
    class="video-js"
    controls></video-js>
    <script src="https://players.brightcove.net/1752604059001/default_default/index.min.js"></script>


    <!-- custom script -->
    <script type="text/JavaScript">
        var myPlayer;

	videojs.getPlayer('video_1').ready(function() {
            myPlayer = this;
            console.log(bc.VERSION);
            console.log(bc.SKIN);
        });
    </script>
Valeur retournée:

La valeur renvoyée dans ce cas est 5.1.7

Cette valeur provient de la configuration du lecteur, qui peut être trouvée en prévisualisant le lecteur dans le navigateur et en passant index.html à config.json.

PEAU

le bc.SKIN propriété spécifie le skin appliqué au lecteur.

Voici un exemple :

    <video-js id="video_1" style="width: 640px; height: 360px;"
    data-video-id="4029697544001"
    data-account="1752604059001"
    data-player="default"
    data-embed="default"
    class="video-js"
    controls></video-js>
    <script src="//players.brightcove.net/1752604059001/default_default/index.min.js"></script>

    <!-- custom script -->
    <script type="text/JavaScript">
        var myPlayer;

	  videojs.getPlayer('video_1').ready(function() {
            myPlayer = this;
            console.log(bc.VERSION);
            console.log(bc.SKIN);
        });
    </script>
Valeur retournée:

La valeur renvoyée dans ce cas est nul , ce qui signifie que le skin Luna est appliqué à ce joueur. Voici un résumé des valeurs de retour du bc.SKIN propriété:

valeur bc.SKIN Peau appliquée
faux Pas de skin, pour que vous puissiez remplacer avec vos propres styles
graphite Peau de graphite
toutes les autres valeurs (ou leur absence), qui comprend undefined, null, true, etc. Skin Luna (skin par défaut)

Cette valeur provient de la configuration du lecteur, qui peut être trouvée en prévisualisant le lecteur dans le navigateur et en passant index.html à config.json.

Remarque : le bc.SKIN propriété n'est renseignée que si le "skin" property dans la configuration du lecteur a une valeur explicite. Sinon, il sera indéfini.
