Exécution d'une trace de chemin réseau

{{ page.description }}

Présentation

le traceroute L'outil de diagnostic suit le chemin de votre ordinateur vers le site Web en question. L'outil envoie une séquence de paquets sur le réseau et reçoit des réponses de chaque routeur dans le chemin. En faisant cela, il détermine s'il y a des problèmes sur le chemin.

Lorsque vous travaillez avec le support, vous pouvez être invité à leur fournir les informations nécessaires pour tracer le chemin du réseau jusqu'au site Web en question.

Traçage depuis macOS ou Linux

Suivez ces étapes pour exécuter une trace du chemin réseau :

  1. Ouvrez une fenêtre pour l'application Terminal.
  2. Saisissez la commande suivante avec le site Web en question. 
    traceroute your_domain.com

    Si vous savez que votre réseau utilise le protocole Internet version 6 (IPv6), entrez la commande suivante :

    traceroute6 your_domain.com
  3. Vous devriez voir le chemin de trafic emprunté vers votre site. Il sera construit au fur et à mesure que les réponses sont reçues des routeurs en cours de route. La première ligne représente votre routeur. La ligne suivante provient de votre FAI, et elle continue jusqu'à ce que les paquets atteignent votre site Web. Votre sortie doit ressembler à ceci :
  4. Ne vous inquiétez pas de comprendre la sortie. Dans la fenêtre de votre terminal, sélectionnez Éditer > Tout sélectionner. Ensuite, copiez la sortie dans votre presse-papiers.
  5. Collez la sortie dans un e-mail et envoyez-la au personnel de support approprié.

Traçage depuis Windows

Procédez comme suit pour exécuter une trace de chemin réseau :

  1. Ouvrez le Début menu et sélectionnez Courir.
  2. Type cmd et sélectionnez D'accord.
  3. Cela ouvrira l'invite de commande. Entrez la commande suivante avec le site Web en question. 
    tracert your_domain.com

    Si vous savez que votre réseau utilise Internet Protocol version 6 (IPv6), entrez la commande suivante :

    tracert -6 your_domain.com
  4. Vous devriez voir le chemin de trafic pris sur votre site. Il sera construit au fur et à mesure que les réponses sont reçues des routeurs en cours de route. La première ligne représente votre routeur. La ligne suivante provient de votre fournisseur de services Internet et continue jusqu'à ce que les paquets atteignent votre site Web.
  5. Ne vous inquiétez pas de comprendre la sortie. Copiez la sortie dans votre presse-papiers.
  6. Collez la sortie dans un e-mail et envoyez-la au personnel de support approprié.
