Affichage de la configuration

Suivez ces étapes pour afficher la configuration d'un lecteur:

  1. Dans Studio, accédez au JOUEURS module.
  2. Cliquez sur la vidéo dont vous souhaitez visualiser la configuration.
  3. Clique le Publier et intégrer bouton.
  4. Assurez-vous que toutes les modifications sont publiées.
  5. Clique sur le URL du joueur (comme indiqué ci-dessous). Vous verrez le lecteur s'ouvrir dans un navigateur.
  6. Vérifiez l'URL dans le navigateur, il apparaîtra quelque chose comme ceci: 
    https://players.brightcove.net/1752604059001/SAJo2wjVo_default/index.html
  7. Changer la index.html dans l'URL vers config.json.
  8. L'URL modifiée affichera la configuration du lecteur:
    Remarque : Le JSON résultant peut ne pas s'afficher aussi bien que le montre la capture d'écran. Vous pouvez trouver une extension/un module complémentaire de formateur JSON pour votre navigateur afin de vous aider à formater le JSON.

Changer la configuration

Bien que la modification de la configuration dépasse le cadre de ce document, quelques commentaires à ce sujet sont appropriés.

  • Studio fournit une interface utilisateur pour modifier de nombreux aspects de la configuration du lecteur dans le JOUEURS module.
  • Pour un contrôle complet de la configuration du lecteur, le API de gestion des joueurs peut être utilisé. Attention, c'est pour le développeur orienté et utilise boucle déclarations largement. le Pas à pas: Gestion des joueurs Le document fournit une introduction au processus.
